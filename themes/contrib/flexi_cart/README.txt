# Flexi Cart Theme

This a Drupal Commerce Mobile first Theme. Our Commerce theme is the best way for you to display and sell your products. It is designed to make the your products to sell fastly and easily in online. We have provided clear code and comments for theme. Using SASS variable it apply globally.

Font Awesome icons used for browser friendly. Banner/Slider, Social share, regions etc.,  are configurable. To use this module you must have Drupal Commerce 2.x installed.

### Requirements
* Drupal 8.x (https://www.drupal.org/docs/8/install)
* Drupal Commerce Module (https://www.drupal.org/project/commerce)
* Composer (https://getcomposer.org/)
* Compass (http://compass-style.org/install/)
* Ruby (https://www.ruby-lang.org/en/downloads/)

### Installation
* Setup Drupal 8.x
* composer require drupal/commerce
* drush en commerce
* composer require drupal/commerce_cart_flyout
* drush en commerce_cart_flyout
* Enable required module in commerce package `http://localhost/flexicart/admin/modules`
* Enable the Flexi Cart theme in Appearance `http://localhost/flexicart/admin/appearance`
* Create a store in `http://localhost/flexicart/admin/commerce/config/stores`

### Features
* Mobile first Theme
* 18+ regions
* Smooth scroll animation
* Font Awesome icons
* Fluid layout
* Included Sass & Compass source file
* Colors are stored in Sass variable
    * Well organized Sass
    * Slider/Banner configuration
